Research Themes

Video Compression Techniques

This research focuses on optimizing video compression technique using H.264. The work includes:

  • Spatial resolution based macro block mapping for scalable enhancement
  • Spatial-temporal search algorithm to improve prediction accuracy
  • Independent rate control algorithm for efficient bit allocation

Video Analytics

Normal and abnormal human activity detection using CNN, RNN, and LSTM. This deep learning project aims to detect criminal behavior in Indian crime contexts using transfer learning.

Signal Processing

Reconstructing coded-mask images from space using entropy maximization on photon energy data captured in FITS format for unique source mapping.

LiDAR Technology

Sensor fusion using Camera and LiDAR for object classification and positioning in autonomous vehicle scenarios using Carla simulator and ML.

Research Projects

Yield Prediction in Vineyards using LiDAR

Research funded by Department of Science and Technology, WOS-B. 2D/3D imagery used to estimate grape yield using deep learning.

Optimized Video Compression for Real-Time Applications

Funded project on scalable coding algorithms to address dynamic bandwidth streaming using H.264 framework.
