Research Themes

ORCID: https://orcid.org/0000-0002-5083-2900

Video Compression Techniques

This research focuses on optimizing H.264 scalable video compression. It includes:

  • Macro block mapping for resolution enhancement
  • Spatial-temporal prediction algorithms
  • Independent rate control for scalable bit allocation

Video Analytics for Public Safety

Focuses on human behavior analysis using CNN + LSTM to detect criminal activity with transfer learning for Indian crime scenarios.

Signal Processing & Image Reconstruction

Using entropy-based decoding of coded-mask images from photon data captured in space, improving astrophysics image clarity via entropy maximization.

LiDAR + Sensor Fusion

Uses LiDAR and camera fusion in Carla simulator for object detection and 3D localization in self-driving vehicle environments.
